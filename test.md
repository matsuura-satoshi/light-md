# LightMD Test Document

This is a test document for **LightMD**, a lightweight Markdown viewer for macOS.

## Features

- High readability with iA Writer-inspired typography
- Lightweight and fast startup
- View-only — no editing
- Customizable themes

## Code Example

Inline code: `swift build` works great.

```swift
import SwiftUI

struct LightMDApp: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}
```
